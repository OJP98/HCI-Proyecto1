Para usar el documento 'regresio_ml' es necesario instalar unas cuantas librerías en python3.

Numpy: pip3 install numpy
Pandas: pip3 install pandas
MatPlotLib: pip3 install matplotlib

Scikit learn para machine learning:

1. Instalar SciPy: pip3 install scipy
2. Instalar sckikit: pip3 install -U scikit-learn
